# Formatting Rules For Final Submission

## Figures

- Number all figures sequentially.
- Put the caption below the figure.
- Example: `Fig. 3. Open interest distribution across strike prices.`
- Every figure must be mentioned in the text.

## Tables

- Number all tables sequentially.
- Put the table title above the table.
- Example: `Table 2. Database benchmark comparison`
- Every table must be mentioned in the text.

## Report Style

- Use academic tone.
- Avoid first-person casual language.
- Keep paragraphs concise and formal.
- Maintain consistent terminology across report, notebook, and PPT.

## PPT Style

- Use bullet points, not paragraphs.
- Add key values beside charts.
- Prefer large visuals over dense text.

## Export Checklist

- Word to PDF formatting preserved
- figure numbers unchanged after export
- table numbers unchanged after export
- references formatted in IEEE style
- notebook exported to PDF cleanly
